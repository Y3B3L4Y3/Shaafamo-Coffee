## Quick Copy Commands

After placing your images in this folder, rename them using these patterns:

```bash
# Example - adjust source filenames to match yours
mv "IMG_1234.jpg" farmer-basket.jpg
mv "IMG_1235.jpg" cherries-closeup.jpg
mv "IMG_1236.jpg" cherries-drying-1.jpg
# ... etc
```

## Image Tips

- **Format**: JPG recommended (or WebP for smaller files)
- **Resolution**: Keep original resolution - Next.js will optimize automatically
- **Orientation**: Both portrait and landscape work well
